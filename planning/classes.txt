Classes:

- Timeline -> start_date: datetime, end_date: datetime, events_list: array[event]
- Event -> start_date, (optional) end_date, description: str
- WarEvent (extends Event) -> actors: list[figure/group], outcome: str, winner(optional): figure/group

- (Interface) Entity -> get_name(), get_affiliation()
- Figure (implements Entity) -> name: str, birth_date: datetime, death_date (optional): datetime, affiliations (optional): list[group/figure], description: str
- Group (implements Entity) -> name: str, creation_date: datetime, dissolution_date (optional): datetime, affiliations (optional): list[group/figure], description: str
- Country (extends Groups) -> neighbours: list[country]
